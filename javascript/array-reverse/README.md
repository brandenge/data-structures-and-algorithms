# Reverse an Array

### Problem Domain

- Write a function called reverseArray
- Reverse the items in the array
- Do not use built-in methods
- Mutate the original array destructively

- Input: an array
- Output: an array

### Test Cases

- An empty array
- An array of length 1, 2, and 3
- An array of length 10+

### Algorithm

- Declare the reverseArray function
- Declare a left variable to track the left index
- Declare a right variable to track the right index
- Declare a temp variable to temporarily hold one of the indexes for the swap operation
